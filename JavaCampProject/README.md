* JDK 16
* Compiler compliance level: 15
* Spring initializr configration: 
  * Project -> Maven
  * Language -> Java
  * Spring Boot -> v2.4.5
  * Dependencies: 
    * Spring Boot DevTools
    * Lombok
    * Spring Web
    * Spring Data Jpa
    * PostgreSQL Driver
